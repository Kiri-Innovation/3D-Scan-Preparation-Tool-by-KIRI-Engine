macOS build steps
=================

These instructions are for anyone building the macOS release from source.

Before building:

1. Install Python 3.12.
2. Install Node.js/npm.
3. Download _AI_Models-vX.Y.Z.zip from GitHub Releases.
4. Extract it into the repository root so this folder exists:

   _AI_Models/

5. Open Terminal in the repository root.
6. If the scripts are not executable, run:

   chmod +x mac/*.sh

Build commands:

   ./mac/setup_env.sh
   ./mac/create_icns.sh
   ./mac/build_backend.sh
   ./mac/package_app.sh
   ./mac/build_dmg.sh
   ./mac/verify_macos_build.sh

Outputs:

   dist/ScanPrep Tool Backend/ScanPrep Tool Backend
   dist/3D Scan Prep Tool.app
   Packaged/3D-Scan-Prep-Tool-macOS-<architecture>.dmg


Unsigned macOS apps may show a security warning. Users may need to right-click
the app and choose Open unless the app is code signed and notarized.
